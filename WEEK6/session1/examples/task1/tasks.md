## Exercise 3

You need to add some more functions to the code which you saw earlier as the example.

You will need to both add a new function *and* adapt the main() to do this.

There are a number of features to add, and you can complete them in any order:

- find the maximum number
- find the minimum number
- find the range of the numbers (max - min)
- find any numbers which are repeated more than once
- find the most common number (mode)
- sort the numbers from smallest to largest
- find the median number (the middle number / average of 2 middle numbers when sorted)

You may be able to re-use earlier functions in later ones.
