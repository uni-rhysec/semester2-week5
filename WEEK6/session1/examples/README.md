# Functions and File IO

You will find class examples in /examples, and the tasks in their folders.

Remember, you will need to navigate the terminal using:

```
# to go into a folder:
cd folder_name

# to go back out of a folder
cd ..
```

You will be compiling .c files into executables:

```
gcc nameOfFile.c -o nameOfExecutable

# then:

./nameOfExecutable
```
