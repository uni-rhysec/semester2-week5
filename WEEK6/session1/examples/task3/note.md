# Quick Note

There are some sample files in /txt_files.

C uses relative pathing, so you can open these by giving the filename as a path:

```
FILE* file = fopen("txt_files/code.txt", "w");
```

You can find and add more files if you prefer.

## The Tasks

If you are not comfortable with file handling yet, start with cmd_files.c as this is a more basic task.
