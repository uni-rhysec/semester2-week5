# Activity 1

## Task 1 - 1_byref.c

This is a simple task where you will be passing by reference to return multiple values from some functions related to
coordinates.

Remember:

```
*variable -> dereferences a pointer and allows you to access the value stored there

&variable -> references a variable and gives you the a pointer to the value
```

Read the code - a set of functions to init and display the grid is provided and should work. Try plotting a few points
to ensure you understand how this works.

Now add the three currently empty functions:

- move point
- reflect point
- swap point

You should test these to be sure they are working.
