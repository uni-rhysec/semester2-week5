# Week 6: Functions in C

Tasks for Week 6, covering functions, recursion, file I/O, and pass-by-reference.

## Session 1: Functions and File I/O

Topics covered:

- Creating and calling functions
- Function prototypes and headers
- Single Responsibility Principle
- Recursion (factorial example)
- File handling (fopen, fclose, fgets, fprintf)

## Session 2: Pass by Value and Reference

Topics covered:

- Variable scope
- Call-by-value vs call-by-reference
- Using pointers to modify caller's data
- Arrays as function arguments
- Unit testing with assert()

## Compiling and Running

```bash
gcc filename.c -o output
./output
```

Or with warnings enabled (recommended):

```bash
gcc -Wall -Wextra filename.c -o output
./output
```

> [!CAUTION]
> Take care to remove any executables before you commit and push changes
> back to GitHub. You can delete them manually or add them to `.gitignore`.
